How does it work?

Robin is an assistive software solution that allows carers to set up and plan daily activities for the ones they are taking care of. Robin will then assist the individual in performing these activities by providing step-by-step guidance, using images, text and audio instructions.

Online dashboard

The carer can access Robin through the website www.Robinassists.me from a computer or any other internet enabled device and log in on their personal Dashboard. There, they can program the person’s tasks for the day, week, or even the entire month, with the option to modify them at any time.

Mobile app

The activities programmed by the carer in the online dashboard are sent to the individual’s Robin mobile application. When it is time to perform an activity, the app will notify the individual through an alert or notification. This will prompt them to open the scheduled activity in the Robin mobile app, which will automatically show every step to follow to complete the activity. Each step will be carefully explained by voice, text, images and video.
